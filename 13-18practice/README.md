# Offline Notes

PWA-приложение для заметок, собранное по методическим материалам практик 13-18 по дисциплине "Фронтенд и бэкенд разработка".

Приложение работает по `HTTPS`, поддерживает офлайн-режим, push-уведомления, отложенные напоминания и обновление интерфейса через `WebSocket`.

## Возможности

- добавление обычных заметок;
- отправка push-уведомления сразу после создания заметки;
- создание заметки с отложенным напоминанием по дате и времени;
- удаление заметок и отмена связанного напоминания;
- работа как PWA с `Service Worker` и `Web App Manifest`;
- просмотр информации о локальном `HTTPS`-сертификате на вкладке "О нас";
- синхронные уведомления между вкладками через `Socket.io`.

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Запустить сервер:

```bash
npm start
```

3. Открыть приложение в браузере:

```text
https://localhost:3000
```

## HTTPS

- если в корне проекта есть `localhost.pem` и `localhost-key.pem`, сервер использует их;
- если этих файлов нет, сервер автоматически создаёт локальный самоподписанный сертификат и хранит его в `cert.json`;
- информация о сертификате доступна на вкладке "О нас".

## Проверка функциональности

1. Открыть `https://localhost:3000`.
2. Разрешить уведомления для сайта.
3. Создать обычную заметку и проверить мгновенный push.
4. Создать заметку с датой и временем и дождаться отложенного push.
5. Удалить заметку и убедиться, что она исчезает из списка.
6. Открыть вкладку "О нас" и проверить, что сертификат отображается.
