## 📌 Описание проекта

Веб-приложение для управления заметками / задачами с поддержкой:

- ✅ Офлайн-режима (Service Worker + Cache API)
- ✅ Установки на устройство как PWA (Web App Manifest)
- ✅ Архитектуры App Shell (мгновенная загрузка интерфейса)
- ✅ WebSocket (Socket.IO) для обновлений в реальном времени
- ✅ Push-уведомлений с возможностью отложить напоминание (snooze)
- ✅ Локального HTTPS для разработки

Приложение сохраняет заметки в `localStorage`, кэширует статические ресурсы и динамический контент, работает без интернета и отправляет уведомления даже при закрытом браузере.

---

## 🚀 Функциональные возможности

| Функция | Описание |
|---------|----------|
| 📝 Создание заметок | Обычные заметки + заметки с напоминанием (дата/время) |
| 📋 Список заметок | Отображаются все заметки, есть кнопка удаления |
| 🔔 Напоминания | Push-уведомление в указанное время, кнопка "Отложить на 5 минут" |
| 🌐 Офлайн-режим | При отсутствии сети страница загружается из кэша |
| 📱 Установка PWA | Можно добавить на рабочий стол (иконка, splash screen, отдельное окно) |
| ⚡ Реальное время | При добавлении заметки — уведомление у всех открытых клиентов через WebSocket |
| 🔕 Управление уведомлениями | Кнопки "Включить/Отключить уведомления" |
