## 🛠 Технологии

### Бэкенд
- Node.js
- Express.js
- CORS
- Swagger (OpenAPI) для документации

### Фронтенд
- React 18
- Axios для HTTP-запросов
- Sass для стилизации
- CSS-анимации и неоновая тема

## 🚀 Установка и запуск

### Предварительные требования
- Node.js (версия 14 или выше)
- npm (менеджер пакетов)

### Запуск бэкенда

```bash
# Перейти в папку бэкенда
cd 2practice

# Установить зависимости
npm install

# Запустить сервер
node app.js
Сервер будет доступен по адресу: http://localhost:3000
Swagger документация: http://localhost:3000/api-docs

Запуск фронтенда
bash
# Открыть новый терминал
# Перейти в папку фронтенда
cd frontend

# Установить зависимости
npm install

# Запустить React приложение
npm start
Фронтенд будет доступен по адресу: http://localhost:3001

✨ Функциональность
Бэкенд API
✅ GET /products - получение всех товаров

✅ GET /products/:id - получение товара по ID

✅ POST /products - создание нового товара

✅ PATCH /products/:id - обновление товара

✅ DELETE /products/:id - удаление товара

Фронтенд
✅ Отображение всех товаров в виде сетки

✅ Карточка товара с изображением, названием, категорией, описанием, ценой и количеством

✅ Создание нового товара (модальное окно)

✅ Редактирование товара

✅ Удаление товара с подтверждением

✅ Неоновый киберпанк-дизайн с анимациями

✅ Адаптивная верстка

📚 API Документация
После запуска бэкенда документация доступна по адресу:
http://localhost:3000/api-docs

Документация включает:

Описание всех эндпоинтов

Схемы данных

Возможность тестировать API прямо из браузера (Try it out)
